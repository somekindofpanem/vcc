## Commands:

- `vcc depu`: Updates all dependencies
- `vcc run`: Runs a project
- `vcc com`: Compiles a file
- `vcc add`: Adds a dependency to Vcc.toml and runs `vcc depu`
- `vcc init`: Initializes an empty VCC Project
